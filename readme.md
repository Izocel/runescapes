# 🪓 Auto‑Clicker & Key‑Press Automation Script

A lightweight Python automation tool that performs **periodic mouse clicks** and **timed keyboard presses** with human‑like randomization.  
Originally designed for repetitive in‑game actions such as mining and stashing, but flexible enough for any workflow that requires automated input.

---

## ✨ Features

- ⏱️ Automated mouse clicks with randomized timing
- 🎹 Periodic keyboard key presses
- 🎲 Human‑like jitter to avoid predictable patterns
- 📘 Fully documented, clean, modular Python code
- 🔁 Runs continuously until manually stopped

---

## 📦 Requirements

Install dependencies:

```bash
pip install mouse keyboard
```
